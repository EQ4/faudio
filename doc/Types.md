
## Audio Types







Name                     | Description
------------------------ | ----------------------------
`int32, float32, string` | concrete types
`A`                      | type variable
`(A, B)`                 | pair of *A* and *B*
`[A x n]`                | vector of A
`~A`                     | signal of *A*
`A ~> B`                 | processor from *A* to *B*
