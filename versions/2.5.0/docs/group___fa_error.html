Typedef Documentation

typedef struct _fa_error_t* fa_error_t

A type implementing the error interface.

If you cast a value to this type, you must assure that it implements fa_error_interface_t, conversely if you cast a value from this type you can assume that it implements fa_error_t.

You can check whether an arbitrary reference is an error using fa_error_check.

Return whether the given value is an error or not.

This function is often used with log as in:

if (fa_check(value)) {
fa_error_log(NULL, value);

}
