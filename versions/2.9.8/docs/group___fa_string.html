Detailed Description

Provides strings.

A string is an immutable sequence of Unicode characters with single-ownership semantics. To convert a string from and to its ordinary C representation, use fa_string_from_utf8, fa_string_to_utf8, or the utility wrappers string and unstring.

For example:

char* cstr = "Hello";
string_t str = string(cstr);
string_t str2 = fa_string_dappend(str, string(" world!"));
printf("%s\n", unstring(str2));
