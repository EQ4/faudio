void fa_fa_initialize ( )

Performs global initialization.

This function must be called exactly once before any other function in the library. A call to fa_fa_terminate() will reset the global state so that fa_fa_initialize() may be called again and so on.

void fa_fa_terminate ( )

Performs global cleanup.

This function may be used to reset the global state as per above. It is not necessary to call this function before the program finishes.
