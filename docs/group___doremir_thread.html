Detailed Description

Cross-platform threads.

Standard low-level constructs such as threads, mutexes and conditions are provided by this module. For more high-level blocking constructs, see the DoremirThreadFuture and DoremirThreadImproving modules. For non-blocking communication see the Atomic modules.
