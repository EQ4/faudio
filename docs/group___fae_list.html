Detailed Description

Immutable list data structure.

  • Fast front access: (empty, single, cons, head, tail).
  • Slow back access (init, last).
  • Slow random access (index, insert, remove).
  • Slow range access: (take, drop, range, insertRange, removeRange)
  • Linear find, map, fold, reverse and filter.
  • Logarithmic sort.
