Detailed Description

Provides various clocks.

As one might expect, a clock is a provider of time. A clock in faudio, however does not usually indicate real-world time, but the current time of an audio computation.

The most accurate clocks are those derived directly from an audio stream. See fa_audio_stream_clock.
