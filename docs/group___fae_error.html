Detailed Description

Error handling types and functions.

Typedef Documentation

fae_error_t

A type implementing the error interface.

If you cast a value to this type, you must assure that it implements fae_error_interface_t, conversely if you cast a value from this type you can assume that it implements fae_error_t.

You can check whether an arbitrary reference is an error using fae_error_check.

fae_error_callback_t

An error handler routine.

Return whether the given value is an error or not.

This function is often used with log as in:
