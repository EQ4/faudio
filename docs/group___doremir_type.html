Detailed Description

Provides audio types.

Type Description
i8 8-bit integer
i16 16-bit integer
i32 32-bit integer
i64 64-bit integer
f32 32-bit floating point
f64 64-bit floating point
() Unit type
(A, B) Pair of A and B
[A x N] Vector of A
{A} Frame of A
