Detailed Description

This modules exports the entire audio engine.

Example usage:

#import <doremir/audio_engine.h>
int main (int argc, char const *argv[])
{
doremir_audio_engine_initialize();
...
doremir_audio_engine_terminate();
return 0;
}

void doremir_audio_engine_initialize ( )

Performs global initialization.

This function must be called exactly once before any other function in the library. A call to doremir_audio_engine_terminate() will reset the global state so that doremir_audio_engine_initialize() may be called again and so on.

void doremir_audio_engine_terminate ( )

Performs global cleanup.

This function may be used to reset the global state as per above. It is not necessary to call this function before the program finishes.
