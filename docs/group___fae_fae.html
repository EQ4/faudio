Detailed Description

This modules exports the entire audio engine.

Example usage:

#import <fae/audio_engine.h>
int main (int argc, char const *argv[])
{
fae_audio_engine_initialize();
...
fae_audio_engine_terminate();
return 0;
}

void fae_fae_initialize ( )

Performs global initialization.

This function must be called exactly once before any other function in the library. A call to fae_fae_terminate() will reset the global state so that fae_fae_initialize() may be called again and so on.

void fae_fae_terminate ( )

Performs global cleanup.

This function may be used to reset the global state as per above. It is not necessary to call this function before the program finishes.
