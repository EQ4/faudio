Merge two events.

The resulting event has all occurences of the given events interleaved.

Parameters
event1Event to merge.
event2Event to merge.
Returns
A new event.
Laws
merge (switch p x y) (switch p y x) = merge x y
merge (merge x y) z = merge x (merge y z)
merge x never       = x
merge x y           = merge y x
