Begin a new session, and retain it for the duration of a call to the given function.

The given function will be called once after the session has created. The session will be ended after the callback function has returned. If an error occurs while starting the session, the error callback is invoked in place of the session callback.

Parameters
callbackFunction to receive the sesssion.
error_callbackFunction to receive eventual errors.
error_data,session_dataData closed over by the callbacks.

Set a callback to be invoked when a status change is detected on the given session (mainly useful for hardware setup changes).

Note that this function will not modify the devices in a session, you have to restart the session to get a fresh snapshot.

Parameters
deviceThe device.
Warning
On OS X this function must be called from the main thread.
