Detailed Description

Provides message dispatching.

Note
This module is fairly low-level, you typically want to use the the Event wrappers fae_event_send and fae_event_receive instead.

Message passing is the process of delivering a message to an address. Any ordered type can be used as an addreses, but typically it is an objet with fast equality comparison such as an integer. Message passing is captured by two interfaces: Sender which provides messages, and Receiver which accepts them. Note that the method names are contrary to the interfaces: you send to a receiver and receive from a sender.

The dispatcher type provides message passing accross thread boundaries: it implements both Sender and Receiver.

fae_message_sender_t

A type implementing the sender interface.

If you cast a value to this type, you must assure that it implements fae_sender_interface_t, conversely if you cast a value from this type you can assume that it implements fae_sender_t.

fae_message_receiver_t

A type implementing the receiver interface.

If you cast a value to this type, you must assure that it implements fae_receiver_interface_t, conversely if you cast a value from this type you can assume that it implements fae_receiver_t.
